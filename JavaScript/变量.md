		<h5>变量</h5>
		<p>
			变量是存储信息的单元，用来存储特定数据类型的数据，程序能在变量中存值和取值；
		</p>
		<ul>
			<li>标识符</li>
				<p>
				可以将定义要素时使用的字符串序列车位标识符，例如变量名、函数名、对象名；
				<ul>命名规则
					<li>标识符只能有字母、数组、下划线和美元符号组成，不能包含空格、标点符号、运算符等其他符号</li>
					<li>标识符第一个字符不能是数字</li>
					<li>标识符不能与关键字名称相同</li>
				</ul>
				</p>
			<li>变量的声明</li>
			<ul>
				<li>声明变量</li>
				`var userName`
				<li>变量赋值</li>
				`var userName = "windy"`
			</ul>			
		</ul>
